Soundanoid

Soundanoid it's a game made in java, it's just like Arkanoid but with different sounds, the main idea it's to create an interesting beat while playing the game. Project still not finished and a little buggy. The repository can be found at github.

Soundanoid é um jogo feito em java. O jogo é igual ao Arkanoid, mas com uma vertente adicional de som. Cada vez que a bola acerta em algum obstaculo faz um som, o som esta relacionado ao objecto a qual a bola acerta. Este projecto ainda não está acabado e ainda se encontra com alguns erros. O repositório pode ser encontrado no github.

  • Name : Soundanoid
  • Language : Java
  • Category : Game
  • Stage of the App : Buggy
  • Repository : github

JWallet

JWallet it's a Java application that I made in my spare time. This application simulates a virtual wallet, after you create it you can deposit and withdraw money and also check the log of transitions. This app it's located in github.

JWaller é uma aplicação feita no meu tempo livre. A linguagem utilizada foi o java. A objectivo da aplicação é simular uma carteira virtual, podendo o utilizador criar uma carteira, adicionar ou retirar dinheiro e verificar o histórico de transações. O repositório pode ser encontrado no github.

  • Name : JWallet
  • Language : Java
  • Category : Economic Management
  • Stage of the App : Finished but still lacks some features
